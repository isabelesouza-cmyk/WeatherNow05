**WeatherNow**

O WeatherNow é um mini portal de previsão do tempo desenvolvido com Next.js 14 e App Router. A aplicação consome a API do OpenWeatherMap para fornecer dados climáticos em tempo real, como temperatura, sensação térmica, umidade e velocidade do vento.

 Tecnologias Utilizadas
- Framework: Next.js 14 (App Router)
- Linguagem: TypeScript/JSX (.tsx)
- API: OpenWeatherMap - (https://openweathermap.org/api)

**Como Configurar a API Key**

Para que o projeto funcione, você precisa de uma chave de API da OpenWeatherMap.

1- Na raiz do projeto, crie um arquivo chamado ".env.local".
2- Adicione a sua chave no seguinte formato:
   NEXT_PUBLIC_WEATHER_KEY= (sua chave aqui)
   
**Para rodar localmente**
   Instalar as dependências:
Abra o terminal na pasta do projeto e execute:

1- npm install
2- npm run dev
3- Abra o navegador e acesse: http://localhost:3000
